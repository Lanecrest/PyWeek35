How to run/play:
1) Extract 'run_game.py' and 'sprites' folder to the same parent directory
2) Use 'requirements.txt' for any needed modules/libraries
	A) The only required package is: arcade==2.6.17
	B) Python version 3.7 or higher is required as well
3) Launch 'run_game.py'
4) The title screen contains instructions to play
	A) The game requires keyboard input only
